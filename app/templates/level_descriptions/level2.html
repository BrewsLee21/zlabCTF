Útoky typu command injection umožňují útočníkovy provádět příkazy operačního systému na serveru.
V následujícím příkladu uživatel zadá cestu k souboru, jehož obsah chce vypsat na výstup pomocí příkazu cat:
import os

path = input("soubor: ")
os.system("cat " + path)

Této implementace se dá zneužít.
V Linuxovém terminálovém prostředí lze provést několik příkazů, pokud jsou správně odděleny.
; - Provede příkazy po sobě bez ohledu na to, jestli byli úspěšné
&& - Provede příkaz pouze pokud byl ten předešlý úspěšný
Dále například ještě ||, |, &, (), {}

Do python kódu popsaného výše by uživatel mohl vložit například:
aaa; whoami
Příkaz, který by server provedl, by potom vypadal takto:
cat aaa; whoami
Tento příkaz by vypsal obash souboru aaa, a poté by vypsal uživatelské jméno uživatele aktuálně přihlášeného na serveru.
